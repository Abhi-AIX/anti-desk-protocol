**System Maintenance for Humans.**

The Anti Desk Handbook is an documentation site designed to help desk workers fix the physical damage caused by prolonged sitting. It treats the human body like a system that requires regular patching, maintenance, and protocol updates.

## The Stack

Built with the speed of a static site and the organization of a documentation framework.

- **Framework:** [Astro](https://astro.build/)
- **Theme:** [Starlight](https://starlight.astro.build/)
- **Styling:** Custom CSS + Minimalist Design System
- **Content:** MDX (Markdown + Components)
